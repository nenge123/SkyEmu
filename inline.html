网站部署法

<emu-skyemu data-path="1.gba">some text 显示内容 <emu-skyemu>
<script type="text/javascript" src="assets/vba/run.js"></script>

data-path:file Path, support zip/rar4/7z/gba/gbc/gb 设置游戏路径,相对当前网站目录,可以是压缩文件,哪怕改了名字的压缩文件照样能行.压缩文件存在多个skyemu只会启动其中一个.

data-version:set file version,if version not save local then updata设置游戏版本,如果本地不一致则更新
